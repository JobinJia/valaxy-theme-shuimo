# a11y Cleanup Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace 5 `<a href="#" @click.prevent="goBack">` anchors with `<button type="button">`, add ARIA labels to icon-only buttons, give the Toc mobile toggle a proper disclosure pattern, label the Header `<nav>`, and backfill `type="button"` on remaining buttons.

**Architecture:** One commit per file (or per logical change pair). Visual continuity preserved via local SCSS reset on each `&__back` rule (border / background / padding / cursor). One new locale key (`shuimo.nav.label`). All `:aria-label` values reuse existing i18n keys from the i18n cleanup PR.

**Tech Stack:** Vue 3 `<script setup>`, `vue-i18n` v11, SCSS scoped styles, `vitest` for the locale parity test.

**Reference spec:** `docs/superpowers/specs/2026-05-15-a11y-cleanup-design.md`
