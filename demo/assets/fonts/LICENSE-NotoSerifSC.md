# Noto Serif SC — subset

`noto-serif-sc.card-subset.ttf` is a **subset** of **Noto Serif SC (Regular)**,
containing only the glyphs used by this demo's share-card titles and colophon
(~73 characters). It is used at build time by `shareCard.titleFontPath` to render
OG card titles in Node (see the theme README "分享卡片" section).
