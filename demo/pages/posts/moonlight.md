## 月色如水

月既不解饮，影徒随我身。暂伴月将影，行乐须及春。

夜深人静时，打开编辑器，窗外月光洒在键盘上，指尖敲击的声音仿佛也有了诗意。写代码与写诗，本质上都是一种创造——将无序的思绪，编织成有序的篇章。

## 代码如诗

```typescript
function moonlight(opacity: number) {
  return `rgba(255, 255, 240, ${opacity})`
}

const night = {
  sky: 'ink-dark',
  moon: moonlight(0.9),
  shadow: 'follows silently',
}
```

好的代码像好的诗歌，字字珠玑，行行有意。不多一行冗余，不少一处关键。

## 夜阑

我歌月徘徊，我舞影零乱。醒时同交欢，醉后各分散。

永结无情游，相期邈云汉。
