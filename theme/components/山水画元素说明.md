# 山水画元素说明

## 画面构成

生成的山水画遵循传统中国山水画的构图法则，包含以下元素：

### 1. 🏔️ 远山（背景）
**位置**：画面上部
**特征**：
- 多层叠加（5-7层）
- 颜色较淡（淡灰色）
- 轮廓简洁
- 表现远山朦胧的效果

**类型**：
- **最远背景山**：最淡，表现极远处的山脉
- **中远景山**：稍深一些，增加层次感

### 2. 🗻 主山峰群（中景）
**位置**：画面中上部
**特征**：
- 6-8座主要山峰
- 带有纹理（皴法）
- 有树木植被
- 是画面的主体

**包含**：
- **中景山群**：2-3座，分散布局
- **主峰**：最高、最醒目的山
- **左右侧山**：左右呼应
- **台地山**：平顶山，1-2座

### 3. 🌊 水面波纹
**位置**：画面中下部
**特征**：
- 横向的波浪线条（3-4层）
- 由近及远逐渐变淡
- 表现江河湖泊

**说明**：
在传统中国山水画中，水面常用**横向波纹线**来表现。这些线条：
- ✅ 不是错误或瑕疵
- ✅ 是传统的水面表现技法
- ✅ 类似书法中的"波浪纹"笔法

**参考**：
```
传统山水画水纹表现法：
═══════════  ← 第一层波纹（近）
  ～～～～～  ← 第二层波纹
    -----    ← 第三层波纹（远）
```

### 4. ⛵ 小船
**位置**：水面上
**特征**：
- 2-3艘小船
- 随机位置和角度
- 简单的轮廓线

**意境**：
- 表现"孤舟蓑笠翁"的意境
- 增加画面的生活气息
- 衬托山水的空旷

### 5. 🪨 岩石
**位置**：画面前景（最下方）
**特征**：
- 分为两类：
  - **近岸岩石**：3-5块，中等大小
  - **前景大石**：2-3块，最大、最醒目
- 有纹理和阴影

**作用**：
- 增强画面的纵深感
- 前景"压阵"，稳定构图
- 遵循"近大远小"的透视原理

天空（留白）

    ↓

## 关于"水面的线"

你提到的**"画的下边有很多线"**就是**水面波纹**。

### 为什么用线条表现水？

在中国传统山水画中：
- 水面**很少用蓝色填充**
- 而是用**墨线勾勒波纹**
- 这是**"以线传神"**的美学理念

### 传统水纹画法：

**类型**：
1. **平静湖面**：稀疏的横线
2. **流动江河**：波浪形曲线
3. **汹涌激流**：密集的旋涡纹

**本画作采用的是"江河水纹"**：
- 横向波浪线
- 由近及远
- 疏密有致

### 参考名画：

可以参考这些传统山水画来理解水纹画法：
- 范宽《溪山行旅图》
- 郭熙《早春图》
- 马远《水图》（专门画水的作品）

## 如果想要更简洁的水面

如果觉得水波纹太多，可以调整代码：

在 `theme/shuimo-core/src/shanshui/water.ts` 中：

```typescript
// 当前：3-4层波纹
const waveCount = 3 + Math.floor(Math.random() * 2)

// 改为：2-3层波纹（更简洁）
const waveCount = 2 + Math.floor(Math.random() * 2)
```

或者在 `theme/composables/useShanShuiPainting.ts` 中：

```typescript
// 减少水面的垂直高度
elements.push(water(width / 2, waterY, 400, {
  wid: width * 1.2,
  hei: 40,  // 从 60 改为 40
}))
```

## 总结

✅ **水面的线条是正常的、传统的表现技法**
✅ **不是 bug 或错误**
✅ **如果觉得太密集，可以调整参数**
✅ **这就是中国水墨画的特色**

如果你想要更现代的风格（比如蓝色的水面），那需要完全不同的实现方式。当前的实现遵循的是**传统水墨山水画的规范**。
